# FLEETCOR TECHNOLOGIES — Website

Premium futuristic single-page website for **FLEETCOR TECHNOLOGIES PRIVATE LIMITED**.

## Stack
- React 18 + Vite (JavaScript)
- Tailwind CSS
- Framer Motion
- React Hot Toast
- React Scroll
- Lucide React + React Icons

## Run

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```

## Structure

```
src/
  components/   # Reusable UI (Navbar, Logo, Modal, etc.)
  sections/     # Page sections (Hero, Services, About, ...)
  hooks/        # Custom hooks
  data/         # Static content
  assets/       # Images & static assets
```
